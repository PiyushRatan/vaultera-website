# Vaultura — Smart Drawer Security

Vaultura transforms ordinary drawers into secure smart vaults using modern access methods like smartphone unlock, fingerprint authentication, and NFC.  
Designed for privacy, convenience, and everyday security — without complicated installations.

---

## Website Overview

Vaultura is a product-focused business website showcasing a smart drawer security solution.  
The site highlights product features, usage scenarios, pricing tiers, privacy-first design, and a smooth, modern user experience.

**Live Website:** https://vaultera.in

---

## Key Features Showcased

- Smart drawer locking system
- Smartphone-based unlocking
- Fingerprint & NFC access
- Local authentication (no cloud dependency)
- Privacy-first architecture
- Multiple security plans for different needs
- Simple installation, no professional tools required

---

## UI / UX Highlights

The website is built with a strong focus on premium UI and smooth UX:

- Scroll-aware floating navigation
- Section-based snap scrolling
- Active section highlighting
- Subtle micro-interactions and transitions
- Animated pricing comparisons
- Soft confirmation states on interactions
- Clean dark-mode visual language
- Responsive layouts for all screen sizes
